# To-Do-List app Ionic con Angular 
Aplicación movil creada con Ionic y Angular, la cual permite crear tareas, crear categorias, asignar categorias a las tareas y activar o desactivar funcionalidades usando remote-config de Firebase, ademas usa arquitectura limpia.

Tareas
La aplicacion permite crear tareas asignando titulo, descripción y categoria, ademas permite editarlas y eliminarlas, tambien permite marcarlas completadas.

Categorias
La Aplicacion permite crear categorias, editarlas y eliminarlas.

Remote config
Se integro la aplicacion con firebase para activar o desactivar el botón de categorias remotamente.

Clean code 
La aplicacion se organiza en 3 capas Dominio infraestructura y UI 

Requisitos 
Node.js 16+
Ionic CLI
Angular CLI 
Cuenta firebase

Para correr la aplicacion en Android se debe usar capacitor, ya que las nuevas versiones de Angular no son compatibles con Cordova
